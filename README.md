# Таблица значений регистров на MSDOS
Проект предназначен для просмотра текущих значений регистров на MSDOS

## Описание
Данная программа вставляет резидентную программу в стандартную функцию прерывания MSDOS int 09h, которая при нажатии соответсвуюших горячих клавиш выводит, обновляет или закрывает таблицу с значениями регистров и флагов в шестнадцатиричной системе счисления на момент вызова рамки. 
### Горячие клавиши
Ctrl + S - вызов и обновление рамки
Ctrl + F - зыкрытие рамки
## Заметки
Сохранение изменений в видеопамяти в области рамки основано на тройной буферизации, в которой участвуют три следующих буфера: save_buffer, draw_buffer и массив видеопамяти. Save_buffer хранит в себе область видеопамяти, ограниченная рамкой, которая будет выведена после закрытия рамки. Draw_buffer отображает в себе состояние видеопамяти в области рамки на последний момент сохранения. Буферизация реализуется по следующему алгоритму:
1. Программа сравнивает соответственные значения байт в видеопамяти и draw_buffer
2. При обнаружении различий значение из видеопамяти копируется в draw_buffer и save_buffer, тем самым поддерживая актуальность save_buffer
